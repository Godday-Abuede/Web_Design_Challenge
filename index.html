Latitude vs X

What's the weather like as we approach the equator?To know this, we collected data from OpenWeatherMap API and created python script to visualize the weather of 500+ cities across the world of varying distance from the equator.

And we built a series of scatter plots using Matplotlib in order to showcase the relationships between Latitude and Temperature (c), Humidity (%), Cloudiness (%) and Wind Speed (mph).
